# 🏃‍♂️ Riga Rimi Marathon Analysis 🏃‍♂️
Every May, the streets of Riga fill with runners. The Rimi Riga Marathon is one of the Baltic's biggest mass-participation events — but behind the finisher medals and the cheering crowds lies a dataset that tells a richer story. This dashboard shows an interactive data story analysing different trends over time. Developed by [Alvaro Ager](https://www.linkedin.com/in/alvaroager/)
